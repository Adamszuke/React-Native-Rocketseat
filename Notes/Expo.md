# Expo

Expo is an open-source platform for making universal native apps for Android, iOS, and the web with JavaScript and React.

## Run

``` 
npx expo start
```


## Expo config

```
npx create-expo-app@latest --template

    Blank
    Blank (Typescript)
>   Navigation (Typescript) //Select this one
    Blank (Bare)
```
